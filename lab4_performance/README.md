# Лабораторная работа 4. Анализ производительности БД

## Цель работы
Изучение методов анализа и оптимизации производительности запросов в PostgreSQL.

## Используемая база данных
База данных интернет-магазина.

## Выполненные задачи
- Генерация больших объемов данных (20 000+ записей)
- Анализ планов выполнения запросов с помощью EXPLAIN ANALYZE
- Оптимизация запросов путем создания индексов
- Сравнение производительности до и после оптимизации
