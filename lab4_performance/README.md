# Лабораторная работа 4. Анализ производительности БД

## Цель работы
Изучение методов анализа и оптимизации производительности запросов в PostgreSQL.

## Используемая база данных
База данных интернет-магазина.

## Выполненные задачи
1. Генерация больших объемов данных (20 000+ записей)
2. Анализ планов выполнения запросов с помощью EXPLAIN ANALYZE
3. Оптимизация запросов путем создания индексов
4. Сравнение производительности до и после оптимизации
5. Тестирование различных вариантов запросов
6. Проверка работы триггеров
7. Анализ производительности через системные представления

## Файлы лабораторной:
- `data_generator.sql` - генератор тестовых данных
- `explain_before.sql` - анализ до оптимизации
- `explain_after.sql` - анализ после создания индексов
- `01_audit_select.sql` - запросы к системным таблицам
- `02_trigger_test.sql` - тестирование триггеров
- `03_explain_variants.sql` - сравнение разных вариантов запросов
- `04_comparison_analysis.sql` - итоговый анализ сравнения
