# Лабораторная 1
Подписка. (вариант 15)
Имеются подписчики (ФИО, адрес) и издания (индекс, название, месячная стоимость). Подписчик может подписаться на несколько изданий, причем на каждое на определенное число месяцев.

Выходные документы:
Выдать список подписчиков указанного издания, упорядоченный по фамилиям.
Выдать список подписчиков и их изданий с подсчетом суммы подписки каждого подписчика, упорядоченный по фамилиям.

# Лабораторная работа 2. Инсталляция БД и SQL

## Цель работы
Развернуть базу данных PostgreSQL на основе ER-диаграммы, создать таблицы, заполнить их данными и выполнить SQL-запросы.

## Используемая СУБД
PostgreSQL 15+

## Структура базы данных
База данных интернет-магазина включает следующие таблицы:
- users — пользователи
- products — товары
- orders — заказы
- order_items — позиции заказов

## Выполненные задачи
- Создание физической модели БД
- Написание DDL-запросов
- Заполнение таблиц данными
- Выполнение SELECT-запросов с JOIN
- Экспорт дампа схемы БД

## Нормализация
Структура базы данных нормализована до третьей нормальной формы (3НФ):
- отсутствуют повторяющиеся группы (1НФ)
- отсутствуют частичные зависимости (2НФ)
- отсутствуют транзитивные зависимости (3НФ)
