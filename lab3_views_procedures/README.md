# Лабораторная работа 3. Представления и процедуры

## Цель работы
Освоение механизмов абстракции данных с помощью представлений и хранимых процедур в PostgreSQL.

## Используемая база данных
Интернет-магазин (таблицы users, products, orders, order_items).

## Выполненные задачи
- Создание представлений для выходных документов
- Реализация сложных запросов через представления
- Разработка хранимых процедур с параметрами
